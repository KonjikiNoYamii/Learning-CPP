PHASE 1 — PROGRAMMING LOGIC
Duration: 1 – 2 Months
Goal: Melatih otak berpikir seperti programmer

What to Learn
• Condition (if, else, switch)
• Loop (for, while)
• Function
• Basic problem solving

What to Do
• Buat program kecil setiap hari
• Jangan hanya membaca, harus coding

Practice Projects
• Number guessing game
• Calculator CLI
• Palindrome checker
• Prime number finder

Mindset Skill
• Belajar memecah masalah
• Berpikir: input → process → output
